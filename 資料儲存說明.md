# 資料儲存說明

## 歷史紀錄存放位置

歷史結算資料現在儲存在應用程式目錄下的 JSON 檔案中：

### 檔案位置
```
應用程式資料夾/data/history.json
```

範例：
- 開發環境：`D:\DeskTop\R360\MyCalculator\MyCalculator\data\history.json`
- 打包後：`分攤計算器-win32-x64\resources\app\data\history.json`

## 資料備份與同步

### 方式 1：手動備份
1. 複製 `data/history.json` 檔案
2. 存放到安全位置或雲端硬碟
3. 需要還原時貼回即可

### 方式 2：整個資料夾同步（建議）
將整個應用程式資料夾放在雲端硬碟同步目錄中：
- Google Drive
- OneDrive
- Dropbox

這樣所有電腦都會自動同步歷史紀錄！

### 方式 3：使用 Git 同步（進階）
如果你將整個專案推送到 Git：
- 歷史紀錄會隨專案一起版控
- 在不同電腦 `git pull` 即可同步
- 注意：包含個人資料，私有倉庫較安全

## 資料遷移

首次使用新版本時，應用程式會自動將 LocalStorage 中的舊資料遷移到檔案中。

## 優點

✅ 資料與應用程式在同一位置，方便管理  
✅ 可直接複製整個資料夾到其他電腦使用  
✅ 備份、還原簡單直觀  
✅ 適合使用雲端同步工具  
✅ 可以用文字編輯器直接查看/編輯資料

## 注意事項

- 當前帳目和成員設定仍儲存在 LocalStorage（臨時資料）
- 只有已結算的歷史紀錄會儲存在檔案中
- 檔案採用 JSON 格式，可直接用文字編輯器查看
- 刪除 `data/history.json` 會清空所有歷史紀錄（請謹慎操作）
