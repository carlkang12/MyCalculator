# 如何將此程式轉換為 .exe 執行檔

### 1. 準備環境

請在電腦上建立一個新資料夾，並依序建立以下三個檔案：

- `package.json` (設定檔)
- `main.js` (視窗控制腳本)
- `app.html` (這是我們之前的 React 代碼，我已幫您包裝成單一 HTML 格式)

### 2. 安裝與打包 (在終端機輸入)

```
# 1. 安裝必要工具
npm install

# 2. 測試執行 (看功能是否正常)
npm start

# 3. 打包成 EXE 檔案
npm run dist
```

### 3. 完成

打包後的 `.exe` 會出現在 `dist/` 資料夾中。

### 常見問題：

- **為什麼檔案很大？** 因為 Electron 會包含一個微型的瀏覽器核心，確保在任何電腦開啟都長得一模一樣。
- **資料會不見嗎？** 目前這個版本使用記憶體儲存，關閉程式後歷史紀錄會消失。如果您需要長久保存（即使關機後還在），可以再告訴我，我幫您加上「存檔功能」。
